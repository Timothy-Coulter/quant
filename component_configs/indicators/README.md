# Indicator templates

`IndicatorConfig` YAMLs registered with `IndicatorFactory`.

- `ema_fast.yaml`: 12-period EMA for short-term trend.
- `bollinger_wide.yaml`: 20/2.5 Bollinger Bands for volatility regimes.
- `rsi_momentum.yaml`: 14-period RSI with 70/30 thresholds.
- `macd_default.yaml`: Classic 12/26/9 MACD with momentum tagging.
- `atr_trailing.yaml`: ATR-based volatility lens for stop placement.
